🔄 从 Steam 导入游戏

✅ 无需 API Key — 直接从 Steam 客户端本地文件读取你的游戏列表

📁 需要选择的文件：

Steam 在你的电脑上保存了游戏库数据。请找到并选择以下文件之一：

📍 文件位置：
Steam 安装目录 → userdata → <你的SteamID3> → config
📄 localconfig.vdf ← 推荐
路径示例：C:\Program Files (x86)\Steam\userdata\12345678\config\localconfig.vdf
包含你所有游戏的 App ID，部分还含名称
📄 sharedconfig.vdf ← 备选
路径示例：C:\Program Files (x86)\Steam\userdata\12345678\7\remote\sharedconfig.vdf
📂
点击选择 或 拖拽 .vdf 文件到这里
支持 localconfig.vdf / sharedconfig.vdf
🔑 需要 Steam Web API Key — 在这里免费获取（填入任意域名如 localhost 即可）

此方法通过 Steam Web API 获取你的游戏列表，数据更完整（包含游玩时长等信息）。 但由于浏览器跨域限制，如果自动请求失败，可使用下方的手动粘贴方案。
